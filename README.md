# excel-style-chat (엑셀 스타일 채팅)

## 소개
뒷자리에 앉은 무능한 본부장이나 대표이사의 하수인을 자처하는 과장, 신경질적인 강사의 눈을 피해 친구와 채팅을 하기란 쉽지 않습니다. 폰으로 하려니 고개를 쳐박아야 하고, PC 카톡은 티가 납니다. 사내 메신저는 좋은 선택일지 몰라도, 너무 오래 켜놓으면 의심스러워집니다.

엑셀 스타일 채팅은 이런 상황을 대처하기 위해 만들어졌습니다. 엑셀, VS코드, 피그마, PDF 등 다양한 테마를 변경해가며 채팅을 해보세요. 뒷자리의 이사진에게는 여러분을 착취하고 있다는 만족감을 안겨줄 것이고, 여러분은 실제로 일하는 것처럼 느껴져서 우울해질 것입니다.

웹으로 만들어졌기 때문에 어떤 디바이스에서든 쉽게 이용하실 수 있습니다. 그 어떠한 프레임워크나 라이브러리(파이어베이스 제외)도 사용하지 않았기 때문에 보안 위협으로부터 안전합니다.
